A falha recebeu o nome de CVE-2007-4559, e trata-se de um erro no pacote tarfile do Python. A falha está no código que usa a função tarfile.extract() não higienizada ou os padrões internos de tarfileextractall(). É um bug de passagem de caminho que permite que um invasor sobrescreva arquivos arbitrários
